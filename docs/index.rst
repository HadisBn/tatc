==============================================
Tradespace Analysis Toolkit for Constellations
==============================================

Overview
========
TAT-C is a mission engineering tool for early-stage (pre-Phase A) analysis and design of Earth science space missions.
This Python language library provides the low-level library to model observing systems and analyze their performance.
